# 🚀 🧭 1. Workflow ديال البداية (Step by Step)

## 🟢 PHASE 1 — Setup (1 إلى 2 أيام)

### 🧱 الهدف:

تحضر المشروع باش يخدم microservices

### دير هاد الشي:

* Setup Git repo (mono-repo ولا multi-repo)
* Install:

  * Docker
  * Docker Compose
* Create structure:

```
/ent-project
  /auth-service
  /user-service
  /course-service
  /file-service
  /ai-service
  /frontend
  docker-compose.yml
```

### ⏱ الوقت:

👉 1 يوم (إذا خدام بسرعة)

---

## 🟡 PHASE 2 — Core Backend (Auth + User) (2 إلى 3 أيام)

### 🔐 Auth Service

* Keycloak setup
* JWT login
* roles (student / prof / admin)

### 👤 User Service

* CRUD users
* roles mapping
* profile data

### ⏱ الوقت:

👉 2–3 أيام

💡 هذا أهم مرحلة (إذا صلحتيها = باقي المشروع ساهل)

---

## 🟠 PHASE 3 — Course + File Service (3 إلى 4 أيام)

### 📚 Course Service

* create course
* list courses
* assign professor

### 📁 File Service (MinIO)

* upload PDF
* download files
* link file ↔ course

### ⏱ الوقت:

👉 3–4 أيام

---

## 🔵 PHASE 4 — Frontend (Angular) (3 إلى 5 أيام)

### 🎯 Pages:

* login page
* dashboard
* courses list
* course detail
* upload course (prof)
* download (student)

### ⏱ الوقت:

👉 3–5 أيام

---

## 🟣 PHASE 5 — Advanced Services (2 إلى 3 أيام)

### 📢 Notification Service

* email / alerts

### 💬 Chat Service

* WebSockets chat

### 📝 Exam Service

* exams + notes

### ⏱ الوقت:

👉 2–3 أيام

---

## 🤖 PHASE 6 — AI Integration (Ollama) (1 إلى 2 أيام)

### features:

* chatbot for students
* explain course
* Q&A

### ⏱ الوقت:

👉 1–2 أيام

---

## 🔴 PHASE 7 — Docker + Deployment (2 أيام)

* docker-compose full system
* run all services together
* optional Kubernetes (if time)

---

# ⏱️ 🧠 TOTAL TIME

👉 Minimum (fast version): **12 – 15 days**
👉 Normal: **2 – 3 weeks**
